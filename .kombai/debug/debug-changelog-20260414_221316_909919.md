# Debug Changelog — 20260414_221316_909919

## Changes Made

### 1. Removed conflicting CDN UMD scripts from index.html
- **File:** `index.html`
- **Change:** Removed 5 CDN `<script>` tags for React, ReactDOM, react-router-dom, supabase-js, and motion. Kept Tailwind CDN since no PostCSS setup exists.
- **Lines removed:** Previously lines 9–13
- **Why:** These libraries are already installed as npm packages and bundled by Vite. Loading them again via CDN caused 628+ CORB (Cross-Origin Read Blocking) errors that kept growing.
- **Revert:** Add these lines back before the `<link rel="preconnect">` tag:
  ```html
  <script src="https://unpkg.com/react@19/umd/react.production.min.js"></script>
  <script src="https://unpkg.com/react-dom@19/umd/react-dom.production.min.js"></script>
  <script src="https://unpkg.com/react-router-dom@7/umd/react-router-dom.production.min.js"></script>
  <script src="https://unpkg.com/@supabase/supabase-js@2"></script>
  <script src="https://unpkg.com/motion@12/dist/motion.js"></script>
  ```

### 2. Fixed infinite re-render loop in useProducts hook
- **File:** `hooks/useProducts.ts`
- **Change:** Replaced `tags = []` default (which created a new array ref every render) with `options.tags ?? []` + `JSON.stringify(tags)` for the `useCallback` dependency.
- **Lines:** ~85–86 (destructuring) and ~167 (deps array)
- **Why:** `tags = []` in destructuring created a new array reference on every render → `useCallback` saw changed dep → recreated `fetchProducts` → `useEffect` fired → infinite fetch loop → loading never ended.
- **Revert:** Restore original destructuring: `const { category, tags = [], sortBy = 'default', searchQuery } = options;` and change deps back to `[category, tags, sortBy, searchQuery]`

### 3. Removed unnecessary isLoading state from App.tsx
- **File:** `App.tsx`
- **Change:** Removed `useState(true)` for `isLoading`, the `Loading` component, and the `setTimeout` in `useEffect`. The app now renders the Router immediately.
- **Why:** The `isLoading` guard rendered `<Loading />` *outside* the Router for 100ms, then re-mounted the entire Router tree — causing a visible blink, React "Node cannot be found" errors, and unnecessary delay.
- **Revert:** Re-add `useState` import, the `Loading` component, and the `isLoading` guard before `return`.
