# Debug Changelog — 20260414_222249_542769

## Changes Made

### 1. Replaced window.history.pushState with React Router's setSearchParams for category navigation
- **File:** `pages/Shop.tsx`
- **Change:** 
  - Changed `const [searchParams] = useSearchParams()` to `const [searchParams, setSearchParams] = useSearchParams()`
  - Replaced `window.history.pushState({}, '', '/shop')` with `setSearchParams({})`
  - Replaced `window.history.pushState({}, '', '/shop?category=${cat}')` with `setSearchParams({ category: cat })`
- **Why:** `window.history.pushState` doesn't trigger React Router's state update, so `useSearchParams` never detected URL changes. Additionally, the app uses `HashRouter`, so pushState URLs without `#` prefix were completely wrong.
- **Revert:** Restore original `window.history.pushState` calls and remove `setSearchParams` from the destructured hook.
